# [NEL]实体卡顿优化 (No Entity Lag)

**The English description is below.**

## 简介

让您告别实体过多造成的卡顿和掉帧

## 功能

- 在非玩家实体数大于 1k 时清除所有非玩家实体
- 令所有实体在附近 10 格内实体数大于 50 时清除这些实体，不会杀死玩家以及村民
- 在玩家动作栏（快捷栏上方）显示实时实体数量
- TNT 附近五格内的 TNT 实体数大于 200 时清除这些 TNT 实体
- 每 180s 显示一次实体数量信息
- 每 600s 清理一次掉落物

---

## Introduction

Say goodbye to lag and frame drops caused by too many entities.

## Features

- Clear all non-player entities when the number of non-player entities exceeds 1,000.
- Clear entities when there are more than 50 entities within 10 blocks, without killing players or villagers.
- Display real-time entity count in the player's action bar (above the hotbar).
- Clear TNT entities when there are more than 200 TNT entities within 5 blocks of TNT.
- Display entity count information every 180 seconds.
- Clean up dropped items every 600 seconds.
